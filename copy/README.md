## Running Locally


```sh-session
git clone https://github.com/fmac/fmac.ng.git
cd fmac.ng
```


Create a `.env` file similar to [`.env.example`](https://github.com/fmac/fmac.ng/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```
